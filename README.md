# HIV Treatment & Prevention Updates

Curated weekly evidence digests and news roundups on HIV drug therapies and long-acting treatment pipelines, compiled by Antonio Flores (HIV / public health, Médecins Sans Frontières — Doctors Without Borders). The digests support rapid review of recent developments in HIV treatment and prevention.

## Contents

- **Weekly digests** — `YYYY-MM-DD_HIV-Drugs-Weekly-Digest`
- **Evidence digests & pipeline summaries** — e.g. `2026-04-18_HIV-Drugs-Evidence-Digest`, `2026-04-18_Long-Acting-HIV-Drug-Pipeline`
- **Monthly overview** — `HIV_Treatment_Prevention_EvidenceDigest_April2026`

Each report is provided in two formats:

- `.html` — formatted, ready to read in any browser
- `.md` — Markdown source, editable in any text editor

## Naming convention

Files are named `YYYY-MM-DD_Descriptive-Title`. Paired `.html` and `.md` files share the same base name.

## Disclaimer

These digests summarize publicly available research for rapid review and are not a substitute for primary sources or clinical guidance.
